Project Overview

Although my focus career wise has been away from Machine Learning and NLP, I have always been passionate about the field and have worked on several projects in my free time to keep my skills sharp.

Using Neural Networks to predict the Sentiment of Amazon Reviews, in order to be used to classify the overall Sentiment and popularity/interests of products and services when mentioned on Social Medias

Used Pytorch to build my Neural Network and used an LSTM layer as is commonly used with Sentiment Analysis. I had Also tried using CNN and RNN with Keras but found that the flexibility of building your Neural Network from scratch with Pytorch helped in getting the best accuracy and performance as compared to Keras

I Was finally able to achieve 90% accuracy in Predictions of Reviews

Key Features

End-to-End NLP & Sentiment Analysis Pipeline

Designed and executed a complete sentiment analysis workflow on over 2 million Amazon reviews, translating a real business problem into a scalable binary classification solution aligned with customer and PR insights.

Advanced Text Pre-Processing & Feature Engineering

Performed extensive NLP pre-processing, including label engineering, class balancing, stopword removal, tokenization, vocabulary indexing, and sequence padding/truncation to prepare raw text data for deep learning models.

Deep Learning Model Development with PyTorch

Built and trained a custom LSTM neural network from scratch using PyTorch, leveraging its flexibility for fine-grained control over architecture, batching, and training—outperforming equivalent CNN and RNN models built with Keras.

Model Optimization, Evaluation & Performance Tuning

Implemented train/validation/test splits, batch loaders, and hyperparameter tuning across embeddings, hidden layers, and epochs, achieving ~90% prediction accuracy and selecting the best-performing model for evaluation.
