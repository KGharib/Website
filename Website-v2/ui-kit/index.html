Dark Theme Rules

  • Use contrast through depth, not bright colour everywhere.
  • Reserve accent colours for action, status, and meaning.
  • Cards should feel like dashboard surfaces, not marketing tiles.
  • Icons should be simple outline icons with consistent stroke weight.
  • Focus rings must be visible on every interactive element.
